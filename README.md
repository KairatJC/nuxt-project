# omlekyn-test
Моя реализация тестового задания с компании Omlekyn Ltd.
[Демо](http://kairat.me/)

## Build Setup
``` bash
# install dependencies
$ yarn install
# or
$ npm install

# serve with hot reload at localhost:3000
$ yarn dev
# or
$ npm run dev

# generate static project
$ yarn generate
# or
$ npm run generate
```
